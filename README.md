**专业的平面设计套版工具** - 创建模板、批量替换、一键生成，让设计工作流更高效

## 🎯 核心价值

**TemplateFlow** 是专为平面设计师和内容创作者打造的套版系统，解决重复性设计工作的痛点：

- 📐 **模板创建**：可视化设计专业版式
- 🔄 **批量替换**：一键更换图片素材
- ⚡ **效率提升**：90%时间节省，专注创意
- 🌐 **Web原生**：无需安装，跨平台使用

---

## ✨ 核心功能

### 🎨 模板编辑器
- **专业画布**：1920x1080 标准尺寸，自由调整
- **图层管理**：完整的Z轴控制，精确元素排序

- **网格系统**：磁性吸附 + 可调节网格，像素级精准
- **批量选择**：框选多元素，高效批量操作

### 🔄 套版处理器
- **JSON驱动**：结构化模板数据，版本控制友好
- **批量替换**：选择元素 → 上传素材 → 一键替换
- **格式支持**：PNG/JPEG/WebP 多格式输出
- **实时预览**：替换效果即时可见
- **属性保留**：位置、尺寸、图层完全保持

### 📦 数据管理
- **模板导出**：完整保存设计结构和样式
- **批量导出**：一次处理多个设计变体
- **数据恢复**：从JSON快速重建完整模板
- **版本管理**：Git友好的文本格式

## 📖 使用指南

### 🎨 创建模板流程
1. **设置画布**：调整尺寸（默认1920x1080）
2. **添加元素**：上传图片素材
3. **布局设计**：使用对齐工具精确定位
4. **图层管理**：调整元素前后关系
5. **导出模板**：保存为JSON文件

### 🔄 批量处理流程
1. **导入模板**：加载之前创建的JSON文件
2. **选择元素**：点击或框选需要替换的元素
3. **上传素材**：批量选择新图片
4. **执行替换**：一键完成所有替换
5. **导出成品**：选择格式并下载

1. 系统概述
   
1.1 什么是 TemplateFlow？

TemplateFlow 是一个基于Web的平面设计套版系统，它将复杂的设计工作分解为两个核心阶段：

模板创建：使用模板编辑器设计可复用的版式布局。

批量生产：使用批量处理器替换模板中的图片素材，快速生成大量设计成品。

这种工作流特别适用于需要保持设计一致性，但内容需要频繁更换的场景，如社交媒体帖子、电商产品图、活动海报等。

1.2 核心工作流

graph LR
    A[模板编辑器] -->|导出 JSON| B[模板数据文件]
    B -->|导入| C[批量处理器]
    D[新图片素材] -->|上传| C
    C -->|导出| E[成品图片]

1.3 系统要求

浏览器：Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
网络：无需联网（本地文件即可运行）
屏幕分辨率：建议 1280x720 或更高，以获得最佳编辑体验

2. 模板编辑器详细指南

文件：canvas-editor.html

用途：创建设计模板，定义元素的位置、大小和层级关系。

2.1 界面布局

模板编辑器界面分为三个主要区域：

区域	功能描述
左侧栏	画布设置、图片上传、元素列表、数据管理
中央画布	可视化编辑区域，显示画布和所有元素
右侧栏	对齐工具、图层管理、显示设置、属性面板

2.2 画布设置

步骤：

在左侧栏找到 “📐 画布设置” 区域。

输入所需的 宽度 和 高度（单位：像素）。

常用尺寸：

Instagram 方图: 1080 x 1080

Instagram 故事: 1080 x 1920

A4 打印: 2480 x 3508

电脑壁纸: 1920 x 1080

点击 “📐 调整画布大小” 按钮。

效果：中央画布区域会立即更新为新尺寸。

2.3 添加和管理元素

2.3.1 添加图片

在左侧栏找到 “🖼️ 添加图片” 区域。

点击 “📁 选择图片文件” 按钮。

在弹出的文件选择窗口中，可以选择一个或多个图片文件。

图片会自动添加到画布上，并随机放置。

特性：

支持批量上传。

大图片会自动缩放至画布的30%以内，防止超出边界。

每个图片元素会自动分配一个唯一的ID（如 element-1）和图层（Z-Index）。

2.3.2 元素列表

在左侧栏的 “📋 元素列表” 区域可以看到所有已添加的元素。

列表按 图层从高到低 排列。

点击列表中的任意项，可以快速选中画布上对应的元素。

点击 “删除” 按钮可以移除单个元素。

2.4 精确控制

2.4.1 选择元素

单选：直接点击画布上的元素。

多选：

按住 Ctrl (Windows) 或 Cmd (Mac) 键，然后点击多个元素。

框选：在画布空白处按住鼠标左键，拖拽出一个矩形框，释放后框内的所有元素将被选中。

2.4.2 移动元素

选中一个或多个元素。

按住鼠标左键拖拽到目标位置。

如果开启了 网格捕捉，元素会自动吸附到最近的网格点。

2.4.3 调整大小

选中单个元素。

拖拽元素四周出现的 白色小圆点（调整手柄）来改变大小。

角落手柄：等比例缩放。

边缘手柄：单向拉伸。

2.4.4 属性面板

当选中 单个元素 时，右侧栏的 “⚙️ 属性面板” 会激活，可以精确输入：

X 坐标：元素左上角的水平位置。

Y 坐标：元素左上角的垂直位置。

宽度 / 高度：元素的尺寸。

技巧：修改数值后，输入框会失去焦点时自动应用更改。

2.5 专业布局工具

当 选中两个或更多元素 时，右侧栏的 “🎯 对齐工具” 区域将被激活。

工具	功能	说明

⬅️	左对齐	所有选中元素与最左边的元素左边缘对齐

↔️	水平居中	所有选中元素的中心点在水平方向上对齐

➡️	右对齐	所有选中元素与最右边的元素右边缘对齐

⬆️	顶对齐	所有选中元素与最顶边的元素顶边缘对齐

↕️	垂直居中	所有选中元素的中心点在垂直方向上对齐

⬇️	底对齐	所有选中元素与最底边的元素底边缘对齐

📏↔️	水平分布	在最左和最右元素之间，均匀分布所有选中元素

📏↕️	垂直分布	在最顶和最底元素之间，均匀分布所有选中元素

🎯	对齐到画布	将所有选中元素在画布中居中

2.6 图层管理

控制元素的遮挡关系（Z轴）。

工具	功能	快捷键

⬆️ 上移	将选中元素上移一层	Ctrl + ]

⬇️ 下移	将选中元素下移一层	Ctrl + [

⬆️ 置顶	将选中元素移至所有元素之上	-

⬇️ 置底	将选中元素移至所有元素之下	-

2.7 网格与捕捉系统

在右侧栏的 “⚙️ 显示设置” 区域：

显示网格：开关画布上的辅助网格线。

网格捕捉：开启后，移动和调整元素时会自动吸附到网格。

网格大小：设置网格的密度，默认为20px。

最佳实践：设计时开启网格捕捉，确保所有元素都对齐，使版面更整洁。

2.8 缩放与视图控制

在画布上方的工具栏：

滑块：拖动可以平滑缩放画布视图（10% - 200%）。

- / + 按钮：步进式缩放。
  
⊡ (适应屏幕)：自动调整缩放比例，使整个画布在视窗内完整显示。

⟲ (重置缩放)：恢复到100%缩放。

快捷键：

Ctrl + 鼠标滚轮：缩放画布。

Ctrl + 0：重置缩放。

2.9 数据导入/导出

2.9.1 导出模板

在左侧栏 “💾 数据管理” 区域，点击 “⬇️ 导出JSON”。

浏览器会下载一个 .json 文件，例如 canvas-1668480000000.json。

此文件包含：

画布尺寸。

所有元素的ID、位置、大小、图层、图片源等信息。

网格设置。

2.9.2 导入模板

点击 “⬆️ 导入JSON”。

选择之前导出的 .json 文件。

画布将完全恢复到导出时的状态。

2.9.3 导出成品图片

在左侧栏 “📸 导出画布” 区域，点击 “🖼️ 导出为图片”。

浏览器会下载一个 .png 文件，这是当前画布的快照。

3. 批量处理器详细指南
   
文件：json-editor.html

用途：导入模板，批量替换图片，生成最终设计成品。

3.1 界面布局

区域	功能描述

左侧面板	画布预览、元素统计

右侧面板	数据导入、元素列表、批量替换、导出选项

3.2 导入模板数据

在右侧面板找到 “📂 数据导入” 区域。

点击 “📥 导入JSON” 按钮。

选择从模板编辑器导出的 .json 文件。

效果：

左侧的 “🖼️ 画布预览” 会显示模板内容。

元素数量 和 画布尺寸 统计信息会更新。

右侧的 “📋 元素列表” 会填充所有可替换的元素。

3.3 理解元素列表

列表显示模板中所有图片元素。

每个元素包含 缩略图、名称 和 尺寸。

点击元素项 可以选中它，选中的元素在画布预览中会显示紫色虚线边框。

3.4 选择要替换的元素

单选：直接点击列表中的元素。

多选：按住 Ctrl 或 Cmd 键点击多个元素。

画布点选：直接在左侧画布预览中点击元素。

提示：选中的元素列表项会有高亮背景。

3.5 批量替换图片

在右侧面板找到 “🔄 批量替换图片” 区域。

点击 “🖼️ 选择替换图片” 按钮。

在文件选择窗口中，选择一张或多张新图片。

点击 “🔄 批量替换图片” 按钮。

替换逻辑：

系统会按照 元素列表中的顺序，依次用你上传的图片进行替换。

如果上传的图片数量少于选中的元素数量，图片会循环使用。

替换后，元素的原始位置和尺寸保持不变，只有图片内容改变。

3.6 导出最终成品

3.6.1 导出画布图片

在右侧面板 “📦 导出选项” 区域，设置导出参数：

包含背景：是否导出白色背景（默认勾选）。

导出格式：选择 PNG、JPEG 或 WebP。

点击 “🖼️ 导出画布图片” 按钮。

浏览器会下载处理后的成品图片。

3.6.2 导出JSON

点击 “💾 导出JSON”，可以保存替换后的新模板数据，方便下次继续编辑。

4. 高级技巧与最佳实践
   
4.1 模板设计原则

预留空间：为替换的图片预留足够的尺寸空间，避免新图片被裁剪。

命名规范：在元素列表中，为关键元素（如Logo、主图、标题）设置清晰的名称，方便后续识别。

使用参考线：利用网格和对齐工具，确保版面整洁、专业。

图层逻辑：将背景元素放在底层，装饰元素放在中层，核心内容放在顶层。

4.2 批量处理工作流示例：制作一周社交媒体内容

模板设计：

在模板编辑器中创建一个包含Logo、产品图占位符、宣传文案占位符的Instagram帖子模板。

导出为 instagram-post-template.json。

素材准备：

准备7张不同的产品图片，命名为 day1.jpg, day2.jpg … day7.jpg。

批量生产：

在批量处理器中导入 instagram-post-template.json。

选中产品图占位符元素。

上传 day1.jpg 到 day7.jpg。

执行批量替换。

导出第一张成品图。

重复操作：

重复步骤3，每次只需替换文案占位符（如果支持文字）或导出不同变体。

4.3 键盘快捷键总览

快捷键	功能	适用场景

Delete	删除选中元素	模板编辑器

Ctrl/Cmd + A	全选所有元素	模板编辑器

Ctrl/Cmd + ]	图层上移	模板编辑器

Ctrl/Cmd + [	图层下移	模板编辑器

Ctrl/Cmd + 0	重置缩放	模板编辑器

Ctrl/Cmd + 鼠标滚轮	缩放画布	模板编辑器

4.4 故障排除

问题	可能原因	解决方案

图片上传后不显示	图片格式不支持或文件损坏	确保是 JPG/PNG/WebP 格式，文件未损坏

导出的JSON文件无法导入	JSON文件被手动修改，格式错误	不要手动编辑JSON文件，或使用JSON验证工具检查

批量替换后图片变形	新图片的宽高比与占位符差异太大	在模板设计时，占位符的宽高比应与最终图片一致

操作卡顿	画布尺寸过大或元素过多	适当减小画布尺寸，或分批处理元素

6. 附录：术语与数据结构
   
5.1 术语表

术语	解释

模板	一个包含布局、元素位置和样式的可复用设计框架。

元素	画布上的一个对象，通常是图片。

画布	进行设计工作的主要区域，有固定的尺寸。

图层	控制元素之间前后遮挡关系的属性（Z-Index）。

网格捕捉	一种辅助功能，使元素在移动时自动对齐到网格线。

JSON	一种轻量级数据交换格式，用于存储和恢复模板数据。

5.2 JSON 数据结构详解

导出的JSON文件是一个对象，主要包含以下字段：



{
  "canvasWidth": 1920,       // 画布宽度
  "canvasHeight": 1080,      // 画布高度
  "gridSize": 20,            // 网格大小
  "snapToGrid": true,        // 是否开启网格捕捉
  "elements": [              // 元素数组
    {
    
      "id": "element-1",             // 元素唯一ID
      "name": "logo.png",            // 图片文件名
      "originalName": "logo.png",    // 原始文件名
      "src": "data:image/png;base64,...", // 图片的Base64编码
      "x": 100,                     // X坐标
      "y": 50,                      // Y坐标
      "width": 200,                 // 宽度
      "height": 100,                // 高度
      "originalWidth": 400,         // 原始图片宽度
      "originalHeight": 200,        // 原始图片高度
      "zIndex": 2                   // 图层顺序
      
    }
  ],
  "timestamp": "2023-11-15T10:30:00.000Z" // 导出时间戳
}


开发者提示：你可以通过编程方式生成或修改此JSON文件，以实现更高级的自动化流程。
