**专业的平面设计套版工具** - 创建模板、批量替换、一键生成，让设计工作流更高效

## 🎯 核心价值

**TemplateFlow** 是专为平面设计师和内容创作者打造的套版系统，解决重复性设计工作的痛点：

- 📐 **模板创建**：可视化设计专业版式
- 🔄 **批量替换**：一键更换图片素材
- ⚡ **效率提升**：90%时间节省，专注创意
- 🌐 **Web原生**：无需安装，跨平台使用

---

## ✨ 核心功能

### 🎨 模板编辑器
- **专业画布**：1920x1080 标准尺寸，自由调整
- **图层管理**：完整的Z轴控制，精确元素排序
- **智能对齐**：9点对齐 + 元素分布，专业布局
- **网格系统**：磁性吸附 + 可调节网格，像素级精准
- **批量选择**：框选多元素，高效批量操作

### 🔄 套版处理器
- **JSON驱动**：结构化模板数据，版本控制友好
- **批量替换**：选择元素 → 上传素材 → 一键替换
- **格式支持**：PNG/JPEG/WebP 多格式输出
- **实时预览**：替换效果即时可见
- **属性保留**：位置、尺寸、图层完全保持

### 📦 数据管理
- **模板导出**：完整保存设计结构和样式
- **批量导出**：一次处理多个设计变体
- **数据恢复**：从JSON快速重建完整模板
- **版本管理**：Git友好的文本格式

## 📖 使用指南

### 🎨 创建模板流程
1. **设置画布**：调整尺寸（默认1920x1080）
2. **添加元素**：上传图片素材
3. **布局设计**：使用对齐工具精确定位
4. **图层管理**：调整元素前后关系
5. **导出模板**：保存为JSON文件

### 🔄 批量处理流程
1. **导入模板**：加载之前创建的JSON文件
2. **选择元素**：点击或框选需要替换的元素
3. **上传素材**：批量选择新图片
4. **执行替换**：一键完成所有替换
5. **导出成品**：选择格式并下载

## 💡 专业技巧

### 🎯 设计技巧
- **网格对齐**：开启20px网格，确保像素级精准
- **图层命名**：为重要元素添加描述性名称
- **尺寸预留**：为不同尺寸的替换素材预留空间

### ⚡ 效率技巧
- **批量选择**：使用Ctrl+点击多选元素
- **快捷键**：Delete删除，Ctrl+A全选
- **模板复用**：保存常用模板为JSON文件

### 🔧 高级功能
- **精确控制**：在属性面板输入具体数值
- **批量应用**：设置一个元素尺寸，应用到所有选中元素
- **磁性吸附**：开启网格捕捉，自动对齐
